# 5 Exemple de Site-uri pentru Reședințe Private

Această colecție conține 5 exemple de site-uri web moderne și profesionale pentru reședințe private, fiecare cu design unic și funcționalități interactive. Toate site-urile folosesc **Tailwind CSS** și **Bootstrap** cu o paletă de culori **galben-gri** și includ meniuri sticky, efecte și animații moderne.

## 🏗️ Tehnologii Utilizate

- **HTML5** - Structura semantică
- **Tailwind CSS** - Framework CSS utility-first
- **Bootstrap 5.3** - Componente responsive
- **JavaScript ES6+** - Interactivitate modernă
- **Font Awesome 6.4** - Iconuri vectoriale
- **Google Fonts** - Tipografie personalizată

## 🎨 Caracteristici Comune

✅ **Meniu sticky horizontal** centrat cu logo în mijloc  
✅ **Paletă de culori galben-gri** consistentă  
✅ **5 pagini principale**: Acasă, Despre, Galerie, Articole, Contact  
✅ **Header și footer** separate cu iconuri sociale  
✅ **Animații și efecte moderne**  
✅ **Design responsive** pentru toate dispozitivele  
✅ **Smooth scrolling** și navigare intuitivă

## 🚀 Instalare și Utilizare

### Cerințe de Sistem:
- Orice server web (Apache, Nginx, sau local)
- Browser modern cu suport JavaScript ES6+

### Pași de Instalare:

1. **Clonează repository-ul**:
```bash
git clone <repository-url>
cd residence-websites
```

2. **Deschide într-un server local**:
```bash
# Cu Python
python -m http.server 8000

# Cu Node.js
npx serve .

# Cu PHP
php -S localhost:8000
```

3. **Accesează site-urile**:
- Site 1: `http://localhost:8000/site1/`
- Site 2: `http://localhost:8000/site2/`
- Site 3: `http://localhost:8000/site3/`
- Site 4: `http://localhost:8000/site4/`
- Site 5: `http://localhost:8000/site5/`

## 📱 Responsive Design

Toate site-urile sunt optimizate pentru:
- 📱 **Mobile** (320px+)
- 📟 **Tablet** (768px+)
- 💻 **Desktop** (1024px+)
- 🖥️ **Large screens** (1440px+)

### Breakpoints Bootstrap:
```css
/* Small devices */
@media (max-width: 575.98px) { }

/* Medium devices */
@media (min-width: 576px) and (max-width: 767.98px) { }

/* Large devices */
@media (min-width: 768px) and (max-width: 991.98px) { }

/* Extra large devices */
@media (min-width: 992px) { }
```

## 🎨 Personalizare

### Culori Principale:
```css
/* Galben */
--primary-yellow: #f59e0b;
--yellow-hover: #d97706;

/* Gri */
--primary-gray: #6b7280;
--gray-dark: #374151;
--gray-light: #e5e7eb;
```

### Modificare Rapidă:
1. Editează variabilele CSS în fișierele `style.css`
2. Înlocuiește gradienții pentru efecte diferite
3. Actualizează fonts în `@import` statements

## 🐛 Depanare

### Probleme Comune:

**1. Animațiile nu funcționează**
```javascript
// Verifică dacă Intersection Observer este suportat
if ('IntersectionObserver' in window) {
    // Cod animații
}
```

**2. Fonts nu se încarcă**
```html
<!-- Adaugă preload pentru fonts -->
<link rel="preload" href="font.woff2" as="font" type="font/woff2" crossorigin>
```

**3. Performance issues**
```javascript
// Folosește debounce pentru scroll events
const debouncedScroll = debounce(handleScroll, 16);
window.addEventListener('scroll', debouncedScroll);
```
